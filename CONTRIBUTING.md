# Contributing to Rocket Shooter
## Getting Started
1. Fork the repo
2. Clone: git clone https://github.com/YOUR_USER/rocket-shooter.git
3. Install: npm install
4. Dev: npm run dev
## Commit Conventions
- feat: new feature
- fix: bug fix
- docs: documentation
- refactor: code change
## Pull Requests
1. Update CHANGELOG.md
2. Ensure tests pass
3. Request review
